## ✨ 你将体验到什么？

不想玩千篇一律的武侠游戏？SMS RPG 让AI扮演"墨言"，为你生成**专属**的江湖故事。

- 📖 你的选择 → 真实的后果
- 💕 NPC会记住你说的话、做过的事
- ⚡ 心结未解？好感度再高也没用
- 💾 5个存档槽位，随时读档

新游戏 1

## 🎯 核心特性

### Drive Engine v3.0

| 系统 | 功能 |
|------|------|
| 目标系统 | 终极→阶段→小目标，自动生成，永远有事做 |
| 挑战系统 | 心结系统：好感度85封顶，需解开心结才能突破 |
| 危机系统 | 每15-25回合触发：道侣被掳、误会加深、情敌上门 |
| 进度追踪 | 修仙/关系/资源/影响力四维可视 |

### 叙事风格

> 通俗、利落、有人味，接近《庆余年》的叙事感

《鹿鼎记》风格的市井描写，让NPC不再是木头人。

### 存档系统

```
./sms-rpg-saves/
├── save_001.json  # 槽位1
├── save_002.json  # 槽位2
├── save_003.json  # 槽位3
├── save_004.json  # 槽位4
└── save_005.json  # 槽位5
```

JSON格式存档，包含完整世界状态：
- 玩家属性（HP/MP/修为/金钱）
- NPC关系网
- 任务进度
- 物品背包
- 剧情摘要

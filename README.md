# SMS RPG - 文字冒险游戏引擎

**版本：v3.0-drive-engine**

一个功能完整的文字冒险游戏引擎，支持存档系统、目标追踪、挑战系统和危机事件。

## ✨ v3.0 新特性：Drive Engine

### 四大核心引擎

```
┌─────────────────────────────────────────────────────────────┐
│                     SMS RPG 引擎架构                          │
├─────────────────────────────────────────────────────────────┤
│  World State Graph  →  世界状态图（实时数据库）              │
│          ↓                                                   │
│  Drive Engine ⭐NEW  →  驱动引擎（目标与挑战）                │
│          ↓                                                   │
│  Genesis Engine      →  创世引擎（叙事生成）                  │
│          ↓                                                   │
│  Canon Guardian      →  正典守护者（一致性检查）              │
└─────────────────────────────────────────────────────────────┘
```

### Drive Engine 核心能力

1. **目标系统 (Goal System)**
   - 终极目标 → 阶段目标 → 小目标 → 日常
   - 自动生成新目标，让玩家永远有事可做
   - 目标进度可视化追踪

2. **挑战系统 (Challenge System)**
   - 心结系统：每个道侣都有需要解开的心结
   - 心结未解时好感度上限85
   - 挑战揭示需要时间，不能一次性解决

3. **危机系统 (Crisis System)**
   - 每15-25回合触发危机事件
   - 道侣被掳、误会加深、情敌出现、仇家上门等
   - 危机有明确时间限制和后果

4. **进度追踪器 (Progress Tracker)**
   - 修仙、关系、资源、影响力四维度追踪
   - 停滞检测（>10回合无进展→警告）
   - 近期成就记录

## 🎮 游戏特性

- **5个存档槽位** - 随时保存/加载进度
- **世界状态管理** - NPC、地点、物品、关系网
- **《鹿鼎记》风格叙事** - 市井、机灵、有趣、通俗有人味
- **千倍返还系统** - 送礼触发千倍返还（可选设定）

## 🚀 快速开始

### 在 Coze/扣子 中使用

1. 安装技能到你的 Bot
2. 发送 "新游戏" 开始冒险
3. 按提示创建角色和世界

### 基本指令

| 指令 | 功能 |
|------|------|
| 新游戏 [槽位号] | 开始新游戏 |
| 继续游戏 [槽位号] | 加载存档 |
| 存档列表 | 查看所有存档 |
| 状态 | 查看当前状态 |
| 存档 | 保存进度 |

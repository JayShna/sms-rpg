# SMS RPG 指令式引擎 v2.0.0

## 重大更新

SMS RPG 从代码式引擎彻底重构为**指令式引擎**。

### 核心变化

| 对比项 | v1.x (旧版) | v2.0 (新版) |
|--------|-------------|-------------|
| 启动方式 | 需要配置API Key | **零配置，直接开始** |
| 运行模式 | Node.js引擎 | Agent直接生成剧情 |
| 存档系统 | 本地文件 | JSON文件，5个槽位 |
| 世界观 | 预设模板 | 完全自定义 |

### 新功能

✅ **零配置启动** — 不需要任何API Key，加载Skill即可开始游戏

✅ **完全自定义世界观** — 武侠、科幻、商战、任何你能想到的世界

✅ **5存档槽位** — 同时保存5个不同角色的冒险

✅ **智能叙事引擎** — Agent扮演"墨言"创世者，维持世界一致性

✅ **回合制选项** — 每轮3-4个行动选项，支持自定义行动

### 使用方法

1. 加载Skill：`skill_load: sms-rpg-instruction`
2. 输入：`新游戏`
3. 按引导创建角色、选择世界观
4. 开始冒险！

### 存档指令

- `存档列表` — 查看所有存档
- `继续游戏 [槽位号]` — 读取存档
- `删除存档 [槽位号]` — 删除存档
